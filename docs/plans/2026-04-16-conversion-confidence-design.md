# Conversion Confidence Design Notes

## Ordering

1. Operator proof log appears after the AI bot builder section because it expands the credibility claim without using fake testimonials.
2. Sample kit appears after the outcome section because users should see the promised artifacts before reading curriculum details.
3. Next steps appears immediately before the contact form because the main hesitation there is "what happens after I apply?"
4. Mobile summary appears in the hero so small-screen visitors can understand the offer without scrolling through long details.

## Copy Direction

- Keep wording concrete: result, step, preparation, sample.
- Avoid broad AI hype.
- Use the operator's actual systems as proof, but translate them into beginner language.

## Interaction

Sample cards use copy buttons so the section is not just decorative. The copied text is a lightweight prompt starter that can be pasted into an AI tool or into the consultation form.
